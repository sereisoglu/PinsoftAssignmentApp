# PAA

> This project was made within "Pinsoft iOS Assignment".

## Architecture

MVVM architecture was used in this project.

## Auto Layout

NSLayoutConstraint and LBTATools are used for AutoLayout.

## Compatibility

Requires iOS 11.0 or later. Compatible with iPhone.
